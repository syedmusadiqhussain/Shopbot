# ShopBot — AI Chatbot Template (Flask + Ollama)

ShopBot is a simple, client-customizable AI chatbot for ecommerce stores. It uses a single-page chat UI plus a Flask backend that integrates with Ollama for local LLMs.

## Demo UI

![ShopBot UI](shopbot-python/Screenshot%202026-04-20%20005944.png)

## Project folder

The runnable app lives in [shopbot-python/](shopbot-python/).

## Run locally

```bash
cd shopbot-python
pip install -r requirements.txt
python app.py
```

Open: http://localhost:5000
